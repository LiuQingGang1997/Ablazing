请在生成 Git 提交信息时遵循以下格式：

1. **类型和简述**：
   - 格式为：`<type>: <description>`（例如：`feat: 添加购物车功能`）
   - `<description>` 必须使用中文。

2. **详细备注信息（必填）**：
   在空一行后，必须包含以下三个信息：
   - **时间戳**：当前提交的时间（如：`时间戳：YYYY-MM-DD HH:mm:ss`）
   - **作者**：提交人的姓名或标识（如：`作者：[你的名字/Trae]`）
   - **修改记录**：详细的中文修改记录，说明改动的原因和具体内容。

### 示例：
```
feat: 更新主页布局和导航栏

时间戳：2023-10-25 14:30:00
作者：Trae AI

修改记录：
1. 调整了主页首屏的上下边距，增加了呼吸感。
2. 将品牌 Logo 从方形图片替换为白底圆形的徽标样式。
3. 放大了首屏标题、副标题和统计数据的字体大小，解决页面看起来较空的问题。
```
